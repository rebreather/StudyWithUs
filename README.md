## 📝 About The Project

이 프로젝트는 투두메이트라는 어플에서 질문/답변 기능을 추가하여 이용자가 좀 더 동기부여를 받을 수 있도록 만든<br/>
공유형 스터디 플래너 웹 페이지 입니다.
<br>

## 👨‍💻 Tech stack
사용한 기술 스택에 대한 간략한 개요입니다.
- Server: NodeJs(Express모듈)<br/>
- DB: MySQL(AWS RDS)<br/>
- Web Design : CSS, EJS

<br/>
